# עסקלי v0.1 - שלד אפליקציה ראשון

זהו שלד ראשוני לאפליקציית ווב מודולרית לניהול עסקים.
השם "עסקלי" הוא שם זמני.

## מה יש בגרסה הזאת

- דשבורד ראשי
- מסך מסמכים
- פרטי חשבונית לדוגמה
- אזור עסק: ספקים, מוצרים ולקוחות
- דוחות והשוואת מחירים לדוגמה
- הגדרות עסק ומודולים
- עיצוב RTL בעברית
- הכנה לחיבור Supabase
- סכמת בסיס נתונים ראשונית

## איך מריצים במחשב

צריך להתקין Node.js.

```bash
npm install
npm run dev
```

אחר כך פותחים בדפדפן:

```text
http://localhost:3000
```

## חיבור ל-Supabase בהמשך

1. פותחים פרויקט ב-Supabase.
2. מריצים את הקובץ:

```text
supabase/schema.sql
```

3. יוצרים קובץ `.env.local` לפי `.env.example`.
4. מחליפים את נתוני הדמו בקריאות אמיתיות למסד הנתונים.

## מסכים קיימים

- `/` - דף בית
- `/login` - התחברות לדוגמה
- `/onboarding` - יצירת עסק
- `/documents` - מסמכים
- `/documents/rami-6515137` - פרטי חשבונית
- `/business` - אזור עסק
- `/reports` - דוחות
- `/settings` - הגדרות

## מה השלב הבא

- לחבר התחברות אמיתית עם Supabase Auth
- לחבר מסד נתונים אמיתי
- להפוך העלאת מסמך לפונקציה אמיתית
- לשמור חשבוניות וספקים
- להוסיף ניתוח חשבוניות והצלבת מחירים
